# MERN PRO

Fullstack web application built with the MERN stack.

## Tech Stack

### Frontend
- React
- JavaScript / TypeScript
- Vite
- Bootstrap

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## Features
- Responsive UI
- API integration
- Authentication system
- Dynamic content rendering
- Fullstack architecture

## Installation

```bash
git clone https://github.com/IceStorm1k/MERN-PRO.git
cd MERN-PRO

Install dependencies:

npm install

Run the project:

npm run dev

```
### About

This project was created for learning and practicing fullstack web development using the MERN stack.
